# Solutions — compounded learnings

One file per verified learning, written by the /compound skill. ≤30 lines,
frontmatter: title, date, category (bug|decision|gotcha|pattern), tags.
Root cause not symptom; Verification line mandatory. Link related entries
with [[wikilinks]] — graphify turns them into graph edges.

Query this store via `graphify query "<topic>" --budget 2000` from the
workspace root (not grep) once the graph exists.
